# 출원서 Bibliographic Form 입력 안내

본 문서는 KEAPS 의 **출원서 (서지정보)** 입력 항목별 체크리스트이다. 명세서 본문 (`spec-draft.md`) 와 별개로 KEAPS 자체 화면에서 입력해야 하는 항목을 정리한다.

> **법률 자문 아님**. KIPO 공식 안내 (`https://www.kipo.go.kr/`) 가 우선한다. 본 문서는 입력 항목의 의미만 안내한다.

---

## 1. 출원인 (Applicant)

| 항목 | 입력값 | 확인 사항 |
|------|--------|---------|
| 출원인 코드 | KEAPS 자동 발급 (첫 출원 시) | 코드 발급 후 메모 보관 — 모든 후속 출원·등록·연차료 납부에 사용 |
| 성명 (한글) | 출원인 본인 명의 | 주민등록과 일치 |
| 성명 (영문) | 한자/영문 표기법에 따라 변환 | 가족관계증명·여권과 일치 권장 |
| 주민등록번호 (개인) 또는 사업자등록번호 (법인) | | KEAPS 가 행정정보 검증 |
| 우편번호 | | |
| 주소 | 우편물 수령 가능 주소 | KIPO 공식 통지 (의견제출통지서 등) 가 본 주소로 발송됨 |
| 전화번호 / 이메일 | | 의견서 처리 등 신속 연락용 |

**출원인 = 발명자 = 1인 (본인 단독)** 인 경우, 본 항목과 §2 발명자 항목이 동일 정보가 된다.

**복수 출원인** (예: 공동 발명, 회사 + 개인) 인 경우, 출원인별 지분율 입력. 본 발명은 단독 출원 가정.

---

## 2. 발명자 (Inventor)

| 항목 | 입력값 |
|------|--------|
| 성명 (한글) | 발명자 본인 |
| 성명 (영문) | |
| 주소 | |
| 국적 | 대한민국 (한국 국적) |
| 출원인과의 관계 | "동일인" 또는 "기타" |

**중요**: 발명자는 실제 기술적 기여를 한 자연인이어야 한다. AI 도구를 사용한 경우에도, 청구된 기술적 사상을 결정한 자연인이 발명자로 등록된다 (현행 KIPO/한국 특허법 기준 — AI 단독 발명자는 인정되지 않음).

---

## 3. 대리인 (Patent Attorney)

| 항목 | 입력값 |
|------|--------|
| 대리인 유무 | "없음 (본인 출원)" 선택 |

본 가이드는 본인 출원을 가정하므로 대리인 정보는 입력하지 않는다.

---

## 4. 발명의 명칭

| 항목 | 입력값 |
|------|--------|
| 한국어 | "표준 QR 리더 호환 공개 계층과 내장 서명 트레일러를 포함하는 단일 QR 페이로드 검증 시스템과 그 방법" |
| 영문 | "System and Method for Offline Verification of a Layered Single QR Payload with a Plaintext Public Prefix and Embedded Signature Trailer" |

본 명세서 §1 과 동일.

---

## 5. 우선권 주장

| 항목 | 입력값 |
|------|--------|
| 우선권 주장 여부 | "없음" |

**해설**: 우선권 주장은 본 출원 직전 12개월 내에 동일 발명을 다른 국가에 먼저 출원한 경우에만 사용. 본 발명은 한국 최초 출원이므로 해당 없음.

향후 PCT 또는 미국·유럽 출원을 계획할 경우, 본 KR 출원의 출원일이 우선일이 된다 (별도 작업, 12개월 내).

---

## 6. 분할출원 / 변경출원

| 항목 | 입력값 |
|------|--------|
| 분할출원 여부 | "해당 없음" |
| 변경출원 여부 (실용신안 → 특허 등) | "해당 없음" |

본 발명은 신규 단독 출원이므로 모두 해당 없음.

---

## 7. 우선심사 신청 (선택)

| 항목 | 입력값 |
|------|--------|
| 우선심사 신청 여부 | "없음" 또는 "신청" 중 선택 |

**해설**:
- 통상 심사: 청구일로부터 약 12~18개월
- 우선심사: 청구일로부터 약 2~6개월. 추가 비용 약 200,000원
- 우선심사 사유: 자기실시 사실, 제3자 실시 사실, 벤처기업 등 카테고리별 사유 코드 입력 필요

본 발명을 즉시 사업화하지 않는다면 통상 심사 권장 (비용 절감).

---

## 8. 출원료 결제

| 항목 | 입력값 |
|------|--------|
| 결제 수단 | 계좌이체 / 신용카드 / 가상계좌 중 선택 |
| 출원료 산정 | KEAPS 자동 계산 (청구항 15개, 명세서 페이지 수 등 기준) |

KEAPS 가 자동 계산한 금액을 즉시 결제하거나 가상계좌로 추후 납부.

---

## 9. 첨부 문서

| 문서 | 첨부 여부 |
|------|--------|
| 명세서 (한국어) | 필수 — KEAPS 입력 또는 별첨 |
| 청구범위 | 필수 — KEAPS 입력 |
| 도면 | 필수 — `figures/rendered/fig{1,2,3,4,8,9}_*.png` (총 6장) |
| 요약서 | 필수 — KEAPS 입력 |
| 위임장 | 본인 출원 시 불필요 |
| 우선권 증명서 | 우선권 주장 시. 본 발명 해당 없음 |
| 출원인 명의변경 신고서 | 해당 없음 |

---

## 10. 본인 출원 vs 변리사 위임 출원 — 빠른 비교

| 항목 | 본인 출원 (본 가이드) | 변리사 위임 |
|------|--------------------|----------|
| 비용 | 출원료 + 등록료만 (~10만원선) | 출원료 + 등록료 + 수임료 (~150~250만원선) |
| 시간 | KEAPS 학습 시간 + 의견서 직접 대응 | 변리사가 처리 |
| 청구범위 정제 | 본 명세서 §9 그대로 사용 (변리사 정제 미실시) | 변리사가 청구항 표현 정제 |
| 거절이유통지 대응 | 본인이 의견서 작성 (60일 이내) | 변리사가 작성·제출 |
| 등록 가능성 | 청구항 표현 미정제로 거절 위험 다소 증가 | 정제로 등록 가능성 증가 |

본 프로젝트의 목적이 **"일반인 + AI agent 가 직접 출원할 수 있음을 입증"** 이라면 본인 출원이 적합하다. **상업적 권리 보호** 가 우선이라면 변리사 위임을 권장한다.

---

## 11. 출원 후 발생 사건별 대응 안내

| 사건 | 대응 |
|------|------|
| 방식심사 통지 (형식 결함) | 통지서 수령 후 지정 기간 내 보정. KEAPS 의 "보정서" 메뉴 |
| 거절이유통지 (실체심사) | 60일 내 의견서 + 보정서 제출. 통상 인용 선행기술과의 차별점을 청구항·명세서로 추가 명시 |
| 등록결정 | 등록료 납부 (3년분). KEAPS 결제 |
| 등록 후 연차료 | 4년차부터 매년 연차료. 미납 시 권리 소멸 |
| 무효심판 / 정정심판 | 등록 후 제3자 또는 본인이 심판 청구 시 대응 |

본 시점 이후 사건은 본 가이드 범위 외. 발생 시 별도 대응 자료 작성.

---

## 12. 결론

본 출원서 작성에 필요한 모든 정보는 위 12개 항목에 정리되어 있으며, 명세서 본문 (`spec-draft.md`) 과 도면 (`figures/rendered/`) 은 KEAPS 입력 준비가 완료된 상태이다. 출원인 본인의 인증서 / 결제 수단 / 주소 정보만 추가하면 출원 가능하다.

`keaps-submission-guide.md` 의 §8 출원 직전 최종 체크리스트를 마지막에 한 번 더 점검할 것.
