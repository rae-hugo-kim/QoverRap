8. 출원 후 OA (거절이유통지) 대응

출원 후 평균 1~2년 내 심사관 OA 발부 가능. 60일 이내 의견서 + 보정서 제출 필수.

  1. OA 수령 시 즉시 확인 사항: 거절이유 (§29(2) 진보성 / §42(4) 명확성·뒷받침 / §47(2) 신규사항 / §29(1) 신규성) 및 인용 선행기술 식별
  2. 의견서 답변 논거: oa-response-defense-memo.md §2 (broad 청1 유지 논거) 참조
  3. 거절이유 청1 단독에 집중 시 fallback 보정: oa-response-defense-memo.md §3.1 (청1') 또는 §3.2 (청1'') 적용 검토
  4. 인용 본문 위치: oa-response-defense-memo.md §4 의 application-draft 본문 위치 표 활용
전략 메모: 청1 broad 거절 시 청2 (검증 방법) + 청7·8·12·13 종속항으로 등록 가능. 청1 fallback 보정은 거절이유 회피 최후 수단. 본 명세서가 fallback 보정안을 본문 §8.1 · §8.1.1 로 모두 뒷받침하므로 §47(2) 신규사항 위반 위험 없음.
