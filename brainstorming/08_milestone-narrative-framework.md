# 08. Milestone Narrative Framework

## Narrative Position
- 핵심 메시지: "전문직을 위협"이 아니라 "존중을 전제로 한 학습/개척"
- 목표: 반감을 줄이고 협업 가능성을 높이며, 실제 작업 증거를 남긴다.

## Sentence Rules
1. 존중 선언
- "전문직 판단은 대체 대상이 아니라 검증 파트너다."

2. 체험 선언
- "우리는 해당 영역의 프로세스를 직접 수행해 이해 범위를 확장했다."

3. 한계 선언
- "최종 법률 판단과 출원 전략은 전문가 검토가 필요하다."

4. 증거 선언
- "아래 로그/체크리스트/실험 계획으로 과정을 공개한다."

## Messaging Template
- Before: "AI로 전문직을 대체할 수 있다."
- After: "AI로 초안 생산성과 탐색 범위를 넓히되, 전문가 검토 게이트로 품질을 보장한다."

## Do and Don't
### Do
- 작업 과정과 검수 지점을 공개
- 실패 사례도 함께 공유
- 협업 가능한 인터페이스(문서/체크리스트) 제공

### Don't
- 직군 폄하 표현
- 단정적 성공 선언
- 법률 해석을 확정 사실처럼 제시

## Evidence Anchors
- 선행기술 조사 로그
- 청구항 초안 변경 이력
- 검수 게이트 통과 기록
- 실험 프로토콜과 실패 기준

## Done Criteria
- milestone 글 2편 모두 본 프레임을 준수
- 금지 표현 0건
- 한계/협업 문장 1개 이상 포함
