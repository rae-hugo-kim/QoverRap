# QoverwRap Brainstorming Workspace

## Purpose
이 폴더는 `특허-콘텐츠 통합 운영 체계`의 실행 문서를 관리하는 단일 작업 루트다.

핵심 축:
1. 특허 블로깅 전략 (SEO/GEO/AEO + 훅 -> 장문 유도)
2. QR 레이어링 특허 실무 (KR 우선 + AI E2E 초안 + 검수 게이트)
3. 마일스톤 서사 (전문직 존중형 개척 프레임)

## Operating Defaults
- 채널: 단일 테크 블로그
- 독자: 기술 실무자 + 창업자
- 문체: 하이브리드 (기술 근거 + 스토리 훅)
- 언어: 한국어 본문 + 영어 요약
- 발행: 주 1편
- 전환 목표: 훅 포스트에서 장문 테크 아카이브로 유도
- 법률 리스크: 보수적 고지 + 검수 게이트 강제

## Workflow Summary
1. 아이디어 입력 -> `10_question-registry.md`에서 질문 분류
2. 포스트 브리프 생성 (`post_frontmatter`)
3. AI 초안 생성 (본문/영문요약/이미지 캡션/FAQ)
4. `review_gate` 4단계 검수
5. 발행 + CTA 삽입
6. 지표 회수 -> 다음 주 캘린더 업데이트

## Done Criteria
아래 조건을 전부 만족하면 본 운영 체계를 완료로 본다.
- 8주 내 8편 발행 완료
- 모든 발행 글에 `post_frontmatter` 적용
- 특허 관련 글 전부에 보수적 고지 + 검수 로그 존재
- QR 레이어링 문서에 실험 프로토콜과 실패 조건 명시
- 모든 훅 포스트에 장문 아카이브 CTA 포함
